# ADR-0004 — Mode Référence

## Contexte
Le besoin métier principal consiste à comparer une feuille modèle à plusieurs variantes.

## Décision
Créer un mode `Référence` piloté par `Compare_Launcher`.

## Conséquences
- Une référence, N comparaisons.
- Synthèse consolidée par feuille.
- Compatible avec le mode historique.
